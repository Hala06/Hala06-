## 🧠 About Me

I'm a 2nd-year Computer Security student at YorkU passionate about building **AI-powered**, **inclusive**, and **secure** digital experiences. My work focuses on real-world impact,  from accessibility tools to crypto data dashboards.

- 🔐 Focus: Computer Security & AI-Driven Frontends
- 🤖 Projects with Gemini API, LangChain, OpenAI, Tesseract.js
- 🌐 Hackathons (GenAi Genesis 2025, GNEC 2025 Spring, Code for Change, Bitcoin Hackathon, Stellar Gateway Quest, Hack The 6, Bolt, Google Maps Platform Awards )
- 🎨 Loves design, storytelling, and human-centered tech

## 🚀 Featured Projects

### 🔊 [GNEC](https://github.com/Hala06/GNEC)
AI-powered Chrome extension that reads text aloud, enlarges cursor, and helps users with visual or cognitive disabilities.
> **Built with:** React, Flask, Gemini API

### ₿ [Bitcoin NetInsights](https://github.com/Hala06/bitcoin-netinsights)
Live dashboard that visualizes Bitcoin mempool congestion, OP_RETURN usage, and drivechain metrics.
> **Built with:** Next.js, TailwindCSS, Framer Motion

### 🏥 [CareSight](https://github.com/Eaarjun/caresight)
AI assistant for elderly patients to simplify and explain complex medical forms using OCR and Gemini.
> **Built with:** React, Tesseract.js, Gemini API

### 🌌 [Stellar Gateway](https://github.com/Hala06/Stellar-Gateway)
Space mission dashboard with voice guidance, built-in AI assistant, and kid-friendly interactions.
> **Built with:** React, LangChain, Three.js
